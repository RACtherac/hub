## Personal Hub & Portfolio

A portfolio site and collection of interactive tools built with React, TypeScript, and a Node/Express backend.

Includes a Warhammer 40K army builder, Pokemon TCG tracker, todo app, rate limiter demo, and more.

**Disclaimer:** All Warhammer 40K images and related content belong to Games Workshop. This project is non-commercial and built for fun.

## Getting started

Start the API backend:

```bash
npm run server:dev
```

Then start the frontend:

```bash
npm run dev
```

The app should now be running at `http://localhost:5173`.
